(constant) productSchema

Mongoose schema for the Product model. This schema represents product data used in the e-commerce platform. It includes fields such as price, category, and stock level. The `isDeleted` flag is used to implement the tombstone pattern for soft deletes.

(async) startRabbitMQ()

Initializes the RabbitMQ connection and sets up the consumer for the Product Service to handle ORDER_SHIPPED events. On receiving an ORDER_SHIPPED event, it attempts to reduce stock for each product in the order. If stock is insufficient for any product, a STOCK_UPDATE_FAILED event is published as compensation. This function contributes to the Saga pattern for distributed transaction coordination.
